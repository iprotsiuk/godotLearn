# Input Actions

InputMap is bootstrapped in code at startup (`Scripts/Bootstrap/InputBootstrap.cs`) so manual editor setup is not required.

## Actions and defaults

- `move_forward`: `W`
- `move_back`: `S`
- `move_left`: `A`
- `move_right`: `D`
- `jump`: `Space`
- `quit`: `Escape` (releases mouse capture)
- `toggle_debug`: `F1`

## Mouse

- Mouse move: look yaw/pitch.
- Left click: capture mouse.
- Escape: release mouse.
